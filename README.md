# Stack Sandbox: REM Stack (React + Express + MySQL)

## Student
Tina Shabestari

## Stack tested
React, Express, MySQL

## Goal
This sandbox app tests how React connects to an Express backend and prepares for MySQL integration.

## What the app does
- Simple signup form (username + password)
- Sends data from React to Express API
- Demonstrates frontend-backend communication

## Setup instructions
1. Clone the repository
2. Checkout this branch:
   git checkout sandbox-tina-rem

3. Install dependencies:

Client:
cd client
npm install

Server:
cd server
npm install

4. Run the app:

Server:
npm run dev

Client:
npm start

## Environment / tools used
- Node.js
- npm
- VS Code

## Notes about setup
Setting up React and Express was simple, but connecting them required using CORS and proper API routes.

## What I learned
- How React sends data using axios
- How Express handles API requests
- Importance of POST vs GET
- Basic project structure for full-stack apps

## Verdict
The REM stack is flexible and beginner-friendly. It is good for small to medium projects but requires manual setup compared to full frameworks.
